11.表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

我们可以设置表格的对齐方式：

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。
|name|111|222|333|444|
|- | :-: | :-: |:-: |-:|
|aaa|bbb|ccc|ddd|eee|
|fff|ggg|hhh|iii|000|

呈现方式：

name 111 222 333 444
aaa bbb ccc ddd eee
fff ggg hhh iii 000
